# CODTECH Java Internship

This repository contains all the tasks completed as part of the CODTECH Java Internship.

## Task 1: File Handling
- Write, read, and append data to a file using Java.

## Task 2: REST API Client
- Java program that consumes a public REST API and displays data.

## Task 3: Multithreaded Chat Application
- Client-server chat application using Java sockets and multithreading.

## Task 4: AI-Based Recommendation System
- Simple recommendation system that suggests items based on user preferences.

## Technologies Used
